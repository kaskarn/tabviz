Function Reference

Complete reference for the tabviz public API. Functions are grouped by purpose; each name links to its detail page.

Entry points

Function Description
tabviz() Main entry point — interactive table with any mix of columns.
forest_plot() Convenience wrapper that auto-builds a forest column from point / lower / upper (or effects).
web_spec() Alias of tabviz(..., .spec_only = TRUE) — return a WebSpec instead of a widget.
split_table() Build a SplitForest from a spec / widget / data frame. 
# Build a table with a forest plot column
tabviz(data, label = "study", columns = list(
  viz_forest(point = "hr", lower = "lo", upper = "hi"),
  col_numeric("hr", "HR"),
  col_pvalue("pval")
))

# Quick forest plot
forest_plot(data, point = "hr", lower = "lo", upper = "hi", label = "study")

Data columns

Function Description
col_text(), col_label() Text / row-identifier columns
col_numeric() Numeric with formatting and abbreviation
col_n() Sample-size column (default thousands separator)
col_percent() Percentage from proportion or 0–100 value
col_currency() Numeric with currency symbol
col_pvalue() P-value with smart formatting & stars
col_interval() “1.20 (0.90, 1.50)” interval text
col_events() “events/n” clinical format
col_range() “min–max” range
col_date() Date with strftime() formatting
col_reference() Truncated text + optional link
col_heatmap() Numeric with interpolated background color
col_progress() Inline progress bar

Inline visual columns

Function Description
col_bar() Per-row bar (no shared axis)
col_sparkline() Inline sparkline (line / bar / area)
col_icon() Icon / emoji per row
col_badge() Colored status badge
col_stars() Star rating
col_img() Inline image

Per-row visualization columns

These render a small chart per row with a shared x-axis.

Function Description
viz_forest() Forest plot column (point + CI)
viz_bar() Horizontal bar chart
viz_boxplot() Box-and-whisker
viz_violin() Violin / KDE

Effect helpers

Function Description
effect_forest() Define an effect for viz_forest(effects = ...)
effect_bar() Define a bar in viz_bar(...)
effect_boxplot() Define a boxplot in viz_boxplot(...)
effect_violin() Define a violin in viz_violin(...)

Column construction

Function Description
col_group() Group columns under a shared header
web_col() Low-level constructor every col_*() delegates to
web_group() Define a row group explicitly

Styling modifiers

Fluent verbs that mutate a WebSpec, htmlwidget, or tabviz_proxy. See Styling.

Function Description
set_row_style() Row-level styling (bold / color / type / indent / …)
set_column_style() Per-cell styling for one column
set_marker_style() Marker color / shape / opacity / size
set_zoom() Zoom and container constraints
set_theme() Apply a theme by name or object
selectable_themes() Curate the in-widget theme switcher

Themes

Preset themes

Function Description
web_theme_cochrane() The package default — Cochrane teal + warm-coral accent, Inter, comfortable
web_theme_lancet() Lancet navy + refined gold, Georgia serif, comfortable
web_theme_jama() JAMA all-B&W, ultra-compact, Arial
web_theme_dark() Catppuccin Mocha-inspired dark canvas
web_theme() Generic custom-theme constructor
package_themes() Named list of every shipped theme (Cochrane / Lancet / JAMA / Dark)

Theme modifiers

Function Description
set_inputs() Tier 1 inputs (brand, accent, series, fonts, status)
set_variants() Per-table variants (density, header_style, first_column_style)
set_spacing() Per-token spacing override on top of the density preset
set_theme_field() Generic deep setter; path walks the theme tree

Annotations

Function Description
refline() Vertical reference line
forest_annotation() Custom shape per study (frontend WIP)

Data operations

Same dispatch on WebSpec / htmlwidget / tabviz_proxy — see Data operations.

Function Purpose
add_column(), remove_column() Insert / remove columns
move_column(), resize_column() Reposition / resize
update_column() Edit column properties
sort_rows(), filter_rows(), clear_filters() Order / filter rows
select_rows(), move_row() Runtime row controls
update_data() Replace the data frame
toggle_group() Collapse / expand groups
set_cell(), set_row_label(), clear_edits() Proxy-only cell edits

Interaction

Function Description
web_interaction() Build an InteractionSpec
web_interaction_full() All capabilities on (default)
web_interaction_minimal() Hover only
web_interaction_publication() Static / print preset
default_interaction_for_theme() Returns web_interaction_full()

Shiny

Function Description
tabvizOutput() / renderTabviz() Standard output / render bindings
tabviz_proxy() Proxy for live updates
splitTabvizOutput() / renderSplitTabviz() Split-table bindings
split_tabviz_proxy() / split_tabviz_select() Split-table proxy

Export

Function Description
save_plot() Export a widget / spec to SVG, PDF, or PNG
save_split_table() Export every sub-plot of a SplitForest
