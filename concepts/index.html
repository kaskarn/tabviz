Two ways to configure a plot

You can pass everything to tabviz() directly:

tabviz(data,
  label = "study",
  columns = list(viz_forest(point = "hr", lower = "lo", upper = "hi",
                            scale = "log", null_value = 1)),
  row_bold = ~ pval < 0.05,
  marker_color = ~ ifelse(upper < 1, "#16a34a", "#64748b"),
  theme = web_theme_jama()
)

Or build it up with fluent modifiers:

tabviz(data,
  label = "study",
  columns = list(viz_forest(point = "hr", lower = "lo", upper = "hi",
                            scale = "log", null_value = 1))
) |>
  set_row_style(bold = ~ pval < 0.05) |>
  set_marker_style(color = ~ ifelse(upper < 1, "#16a34a", "#64748b")) |>
  set_theme("jama")

Both produce the same widget. Pick whichever reads better at the call site.
