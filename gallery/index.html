Gallery

Live tabviz() widgets across nine pages. Every figure renders from the same data-frame-plus-columns API — sort columns, click filter funnels, drag rows or columns, double-click cells to edit, switch themes from the gear icon. Pick the page closest to what you want to build.

Flagship galleries

Long Meta-Analysis

70 studies across three cohorts with cohort-level and overall pooled summaries, paginated across three pages. Exercises the most-recently-shipped surface end-to-end: pagination, the five-token semantic surface (row_fill on the overall row, row_accent on significant), row_type = "summary" for pooled diamonds.

Domain Dashboards

Information-dense dashboards across sports analytics, customer feedback, portfolio performance, product metrics, airline ops, and climate science. Rings, pictograms, sparklines, heatmaps, badges — each lead picks the column types that fit its domain.

Compose Your Own Brand

End-to-end build of a fictional pharma brand (Acme Cardio Research) one named input at a time. Eight progressive steps from neutrals to a wrapped web_theme_acme() constructor, each with the rendered table at that stage so you see exactly which input controls what.

Pictograms & Rings

Every mode of col_pictogram() and col_ring() with realistic per-mode examples — count vs rating, range remapping, per-row glyph selection, layout = “stack”, auto status-palette thresholds, and the row-token cascade reaching into glyph cells. Closes with a six-row survey-feedback dashboard putting them all together.

More galleries

Clinical Examples

JAMA / Lancet-style forest plots, subgroup analyses, multi-arm dose-response, nested hierarchical groups.

Interactivity Tour

Live-behavior showcase: sort, filter, drag-and-drop, double-click editing, theme switcher, WYSIWYG SVG/PNG export.

Feature Showcases

Statistical models, distribution viz, conditional styling, fluent customization, publication templates, and the full-monty smoke test.

LOTR (Easter Egg)

Three editorial-register themes (dwarven, elvish, hobbit) with bespoke pictogram glyphs and tongue-in-cheek source data.

Example sources

Every gallery example is also available as a runnable R script in the package:

# List the curated gallery examples shipped with the package
list.files(system.file("examples", package = "tabviz"), pattern = "^gallery_")

# Run one
source(system.file("examples", "gallery_26_paginate_meta_analysis.R",
                   package = "tabviz"))
