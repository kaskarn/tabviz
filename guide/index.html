Two entry points

tabviz() is the general-purpose constructor. forest_plot() is a thin convenience wrapper for the common case where you only need point + lower + upper.

# Full control
tabviz(data,
  label = "study",
  columns = list(
    col_text("category"),
    col_numeric("value"),
    viz_forest(point = "hr", lower = "lo", upper = "hi"),
    col_sparkline("trend")
  )
)

# Quick forest plot
forest_plot(data,
  point = "hr", lower = "lower", upper = "upper",
  label = "study",
  scale = "log", null_value = 1
)
TipWhen to reach for forest_plot()

Use forest_plot() when the table is “labels + a forest plot + maybe a CI string.” The moment you need extra columns, badges, or column groups, switch to tabviz().
